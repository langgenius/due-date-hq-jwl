# CPA Tools Integration Research + CPA Field Guide Directory

Research + scoping only — no product code. All output under `docs/integrations/cpa-tools/`.

## What landed

- **`landscape.md`** — CPA/accounting software ecosystem mapped in 3 layers (practice management, ledgers, tax prep) + integration shortcuts, each tool rated by integration openness with sources. Key finding: ingestion is already solved by Migration Copilot CSV presets; the real gap is outbound.
- **`integration-recommendation.md`** — ranked bets; the pick = a private/unlisted Zapier "deadline changed" trigger (one build reaches every PM tool, no review gate, glass-box-safe), scoped to the shipped model (per-filing readiness, one-click apply, audit log).
- **`directory-landscape-and-verdict.md`** — sourced competitive analysis of existing CPA-software directories (G2/Capterra — now consolidating; CPA Practice Advisor owner-conflicted; Jason Staats' independent DB the reference competitor) + taxonomy correction (deadline monitoring is not a top-level category) + honest go/no-go (directory/SEO play = not worth it pre-validation; small neutral outreach asset = qualified yes).
- **`cpa-tools-directory.html`** — prototype "CPA Field Guide" directory (independent, quietly owned; domain cpafieldguide.com purchased). Functional categories with authoritative definitions + inclusion criteria, no-pay-to-list methodology block, firm-size filter, brand-colored logo tiles, per-tool screenshot slots, scroll/hover micro-interactions. Style deliberately separate from DueDateHQ brand. Published as an Artifact.

## Notes / open

- Logo tiles are brand-colored monograms (approximate colors), NOT the real trademarked logos; screenshots are empty slots. Real assets need vendor press kits + own captures (CSP blocks external images; UI is copyrighted/login-gated).
- Name-collision risk flagged: Fieldguide.io is a funded audit-AI company.
- Verdict on record: building this as a directory business is a post-validation move; keep the domain parked unless used as a neutral outreach one-pager.
