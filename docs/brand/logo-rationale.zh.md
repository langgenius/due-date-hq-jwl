# DueDateHQ Logo 说明

> 中文版的 logo rationale,与 `brand-book.md` §1 对齐。

## 一句话
**横杠是一摞待办的截止项,错开的那条是刚插进来的新项——也是此刻最该被你看到的那一个。**

## 为什么是图形,而不是字母「D」
DueDateHQ 的产品内部高度**文字密集**——一屏屏的表单、日期、清单、表格。品牌标识刻意走**反方向**:用一个**纯图形**的 mark,而不是从字母「D」出发的字标。两层意图:

- **和产品内部的「重文字」拉开距离**——logo 是图形,界面是文字,各司其职,品牌这一层得以呼吸。
- **中和严肃感**——截止日、合规本就偏正式、偏紧张;一个简洁、友好的几何图形,让品牌更轻、更好接近。

## 形 → 意
- **四条横杠** = 一摞待办的截止项 / 申报,事务所手里一行行的 obligation。
- **错开的那一条** = 一个**新插进来的 item**(刚进队列、位置还没对齐)。也正因为它是新的、还没归位,它就是**该你先看的那一个**——「新插入」是因,「待关注」是果,同一拍讲了两遍。
  - 这和产品本身相互呼应:界面里 highlight 青色 `--color-brand-highlight` `#14C5F6` 正是「新 / 未读」的标记色(未读圆点、新提醒)。所以 mark 里「错开的新横杠」和界面里「青色标记」说的是一套话:**新来的 = 看这里。**
- **App-icon 形态** = navy 圆角方块(`--color-brand-ink`)上的 ivory 横杠(`--color-brand-ivory`);只用 navy + ivory,mark 里不带强调色。splash 上去掉方框,让横杠自己站出来。
- **HQ** 留在字标里,作为低调的 sans 小标签——它是指挥中心(headquarters),不进 mark。

## 关于第一版(v1)
当前是**第一版**,刻意克制,**先作为占位 / 地基**落地。方向已经立住——图形优先、非字母、stack + 插入——但视觉强度还比较弱;比重、错位幅度、是否引入动效或强调色,留给下一版打磨。
