# cpafieldguide.com backlink kit

**Date:** 2026-07-16 · SEO / GTM
**Companion to:** `get-listed-directory-plan-2026-07-07.md` (that doc = DueDateHQ product
listings; this doc = links to the **content site** cpafieldguide.com, whose ranking
bottleneck is domain authority, not on-page).

## The one idea

Nobody links to "another software directory." People link to **data they want to cite**.
cpafieldguide's linkable assets, in order:

1. **/cpa-software-statistics** — market share (2025 AICPA survey), pricing, and API
   openness across 25 tools in one place. This is the pitch target for writers.
2. **/cpa-software-with-open-api** — the only guide that sorts CPA tools by integration
   openness; genuinely differentiated angle.
3. **/compare** — the all-25 table, useful in community answers.

Every pitch below leads with #1 or #2, never with "check out my directory."

## Red lines (carried from the whole program)

- **Disclose affiliation.** The site itself says "maintained by the team behind
  DueDateHQ" — community posts must say the same. One undisclosed drop in r/taxpros
  gets the domain banned there permanently.
- **No astroturfing, no link exchanges, no paid link schemes.** Slow and real.
- **Facts only.** Every number quoted in a pitch must exist on the page pitched.

## Tier 1 — community answers (free, this week)

Where: r/taxpros, r/accounting, r/bookkeeping, accounting Slack/Discord/FB groups
(same venues + same participate-first norms as the 07-07 plan — coordinate, don't
double-post the two sites into one thread).

**When to use:** someone asks "what software should I use / what does X cost /
does X have an API". Answer the question first, in your own words; the link is the
citation, not the message.

### Template A — pricing/comparison question

> Depends on firm size. Rough starting prices: Financial Cents ~$19/user/mo at the
> low end, Karbon and TaxDome in the $59–67/user/mo range; Drake is the classic
> low-cost unlimited option on the tax-prep side. There's a comparison table of ~25
> tools (price + API openness in one view) here: cpafieldguide.com/compare —
> disclosure: my team maintains that guide (we build DueDateHQ), but no vendor pays
> for placement and the numbers are sourced.

### Template B — "does X integrate / have an API" question

> [Answer for the specific tool first — check the tool's page for the current fact.]
> If it helps, we keep a list of every CPA tool sorted by how open its API is
> (self-serve vs gated vs Zapier-only): cpafieldguide.com/cpa-software-with-open-api.
> Disclosure: maintained by my team (DueDateHQ) — vendor-neutral, no pay-to-list.

## Tier 3 — structural (background, no deadline)

- **Cross-link from duedatehq.com** where editorially honest (e.g. the
  works-with-your-stack page can reference the Field Guide's API-openness data).
  One link, in-context — not sitewide-footer spam.
- **Tool vendors themselves**: each of the 25 profiled vendors has a marketing team
  that monitors brand mentions. When a vs/alternatives page ranks, some will link
  to it ("as compared on…"). No action needed — this arrives with rankings.
- **HARO/Qwoted-style source requests** on accounting-software topics: answer with
  a stat from the statistics page. Low volume, compounding.

## What done looks like

Pre-validation bar, not a growth team's: **3–5 real referring domains in 30 days**
(one community answer that sticks + one newsletter citation + the duedatehq
cross-link ≈ done). Track: GSC → Links report, monthly.
