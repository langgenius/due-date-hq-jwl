# Deadlines 工作文档

`/deadlines`（obligation queue + deadline detail）工作流的实现 spec、工程简报
（eng-brief）、roadmap 与 position memo。与其他产品册的编号章节不同，这里是**带日期的
工作快照**：同一主题以日期最新的为准；与代码或 `docs/dev-file/*` 冲突时，以后者为权威。

事后的迭代记录不在这里，写 `docs/dev-log/YYYY-MM-DD-*.md`。本目录的存量文件于
2026-06-10 从 `docs/dev-log/`（原 `_` 前缀文件）迁入，关联的 cluster spec 见
`docs/product-design/migration-copilot/spec-cluster3-migration.md` 与
`docs/product-design/alerts/spec-cluster4-alerts.md`。
