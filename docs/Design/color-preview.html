DueDateHQ — color system

Navy is the only chrome accent (buttons · links · selection · focus). Every status family is a solid · ink · wash · border triad — solids are fills, inks are the colored text (AA ≥ 4.5:1), washes are tinted backgrounds. Warm + aurora are by exception. Light-mode values; toggle to preview against dark.

Neutrals — the cool-gray workbench base
Accent — the only decorative color
Status triads — color appears only on status
Warm surfaces — by exception, never the default
Aurora — decorative, behind the AI “generating” glow
